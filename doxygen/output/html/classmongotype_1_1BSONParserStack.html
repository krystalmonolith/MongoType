Detailed Description

Stack of BSONParserStackItem for storing the BSONObjectParser parse context.

Stores the current state of the parsed BSON objects in a FILO data structure.

Note
"TOS" == "Top Of Stack"
"FILO" == "First-In-Last-Out"
